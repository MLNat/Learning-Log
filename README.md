July 2: Returned to DataCamp going over semi-familiar material covering Python str, int, boolean and lists. Some key things to remember. Print() with rounded brackets; lists with [] square brackets; indexing starts from 0 left to right, -1 right to left and x: just goes to the end. When extracting or editing a subsection of a list the first number is included, but the second number is excluded. E.g. 4:6 is just index item 4 and 5. Negative list would go -5:-3 as you still read along from left, even if indexing is from the right. + functions differently for str, int, boolean and lists - lists just merge if you do that.
